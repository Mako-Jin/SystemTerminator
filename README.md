### 版本管理
|变更类型|触发条件|版本变化|示例|
|---|---|---|---|
|主版本|核心标识变更（资源类型/URL/方法/联系人）|第一位+1，后续归零|1.2.3.4 → 2.0.0.0|
|次版本|主要字段变更（资源名称/值）|第二位+1，后续归零|1.2.3.4 → 1.3.0.0|
|修订版本|次要字段变更（描述/启用状态）|第三位+1，后续归零|1.2.3.4 → 1.2.4.0|
|构建版本|构建字段变更（构建号/部署时间）|第四位+1	1.2.3.4 → 1.2.3.5|
|客户自定义|客户指定版本|采用客户版本	1.2.3.4 → 2.1.0.0（客户指定）|
